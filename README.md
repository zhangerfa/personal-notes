# 二发的个人笔记 (Erfa's Personal Notes)

这是一个基于 AI 驱动的个人知识管理仓库，由 **Aime AI 助手** 辅助记录与维护。本仓库旨在沉淀技术洞察、哲学思考以及日常生活中的灵光一现。

## 📁 目录结构

仓库采用结构化的分类管理方式，并通过 `INDEX.md` 维护全局索引：

- **[技术与工程](./技术与工程/)**：记录关于 AI、工程架构、Skill 开发等技术领域的深度思考与实践。
- **[哲学与思考](./哲学与思考/)**：沉淀关于人生、社会、自我认知的哲学探索。
- **[个人随笔](./个人随笔/)**：记录日常生活中的感悟与随笔。
- **[金融与投资](./金融与投资/)**：分析市场逻辑、投资框架与宏观经济。
- **[INDEX.md](./INDEX.md)**：全局索引文件，记录所有笔记的路径及其核心条目摘要。
- **[skill_docs/](./skill_docs/)**：存放 AI 助手工作流相关的官方说明文档。

## 🤖 工作流与 Skill 机制

本仓库的核心管理能力由 `personal-notes` 技能提供。

### 触发机制
在与 Aime 对话时，通过以下方式即可触发笔记记录：
- 明确使用触发词：“记下来”、“补充笔记”、“加到笔记”。
- 以“**笔记：**”开头发送内容。
- 发送具有记录价值的个人观点或深度思考。

### 交互确认（核心原则）
为了保证笔记内容的准确性与可控性，AI 助手严格遵循以下流程：
1. **禁止静默修改**：任何对笔记或索引文件的改动，AI 必须先生成 **Git Diff** 并展示给用户。
2. **显式确认**：只有在用户查阅 Diff 并明确表示“确认”、“同意”或“OK”后，AI 才会执行物理写入操作。

### 异步同步
- **本地优先**：AI 的写入操作仅作用于本地文件系统。
- **定时推送**：仓库配置有定时任务，每天 **22:00** 自动将当天的更新推送到远端 Git 仓库。
